# MeshONet

MeshONet is a generalizable and efficient operator learning method for **structured mesh generation**. This repository provides example code for training and inference of MeshONet.

## Environment
- **Python**: 3.11.11  
- **PyTorch**: 2.6.0  
- **CUDA**: 12.2  

> Ensure your system has a compatible GPU and CUDA drivers installed.

## Run
python main.py
